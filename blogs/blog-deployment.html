Deployment & DevOps Tools

Published: December 20, 2025

Common deployment and DevOps tools include:

  • CI/CD: GitHub Actions, GitLab CI, CircleCI
  • Hosting & Platforms: Netlify, Vercel, Cloudflare Pages for static sites; AWS, GCP, Azure for dynamic services
  • Containerization: Docker, container registries, and Kubernetes for orchestration

For a static site like this one, hosting on Netlify, Vercel, or static object storage is a simple and cost-effective choice.
