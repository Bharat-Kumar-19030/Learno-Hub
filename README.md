# LearnWeb — Static Educational Site

This simple static site contains a landing page, policy pages, contact page, and three blog posts about full-stack web development tools.

How to view locally:

1. Open `index.html` in your browser (double-click or use your editor's "Open in Browser").
2. Files created:
   - `index.html` — Home
   - `privacy.html` — Privacy Policy
   - `terms.html` — Terms & Conditions
   - `refund.html` — Refund Policy
   - `contact.html` — Contact page
   - `blogs/` — Blog posts
   - `styles.css` — Stylesheet

No backend or domain is required — this is pure static HTML/CSS.
